  • 类型转换

  1. 隐式类型转换：要求等号两边参与运算的操作数的类型必须一致，如果不一致，满足下列条件会发生自动类型转换。
    (1)两种类型兼容
    如int 和double兼容（都是数字类型）
    (2)目标类型大于源类型
    如double>int 小的转大的
  2. 显示类型转换：
    (1)两种类型相兼容 int <–> double
    (2)大的转成小的 double –> int
    语法：(待转换的类型)要转换的值; 上述代码输出的结果为3.33。首先n1乘上1.0转换成了double类型，再除以n2还是double类型，赋值给d也是double的。输出控制{0:0.00}表示有两位小数。
  3. Convert
    类型相兼容的两个变量，可以使用自动类型转换或者强制类型转换。
    类型不兼容时使用Convert的转换工厂进行转换。
    注意:Conver类型转换时如果str为“abc”，转换成int类型会抛出异常。

  • ++ --使用
    ++：分为前++和后++,前++先加1后参与运算，后++先参与运算再加1。
    --：分为前–和后–,前–先减1后参与运算，后–先参与运算再减1。

    对于像++或–这样只需要一个操作数就能完成的运算，称之为一元运算符（单目）。
    + - * / % 这些需要两个或以上才能完成运算的操作符，我们称之为二元运算符（双目）。
    一元运算符的优先级要高于二元运算符。

  • 关系运算符
    用来描述两个事物之间的关系
    > 、< 、>= 、<= 、== 、!=
    由关系运算符连接的表达式叫关系表达式，其结果是bool类型

  • 逻辑运算符
    &&逻辑与
    ||逻辑或
    !逻辑非
    以逻辑运算符连接的表达式叫做逻辑表达式
    逻辑运算符两边一般放的都是关系表达式或者bool类型的值

  • 复合赋值运算符
    int number =10;

+=：number+=20;//number=number+20;
-=：number-=5;//number=number-5;
*=：number*=2；//number=number*2;

顺序结构：程序从Main函数进入，从上到下一行一行执行，不落下一行
分支结构：if if-else
选择结构：if else-if switch-case
循环结构：while do-while for foreach

  • 方法：
    是一堆代码进行重用的一种机制。
    函数的语法：
    [public] static 返回值类型 方法名([参数列表])
    {
    方法体;
    }
    public：访问修饰符，公开的，公共的
    static：静态的
    返回值类型：如果不需要返回值，写void
    方法名：Pascal 每个单词的首字母都要大写，其余字母小写
    参数列表：完成这个方法所必须要提供给这个方法的条件
