注意：配置 maven 环境变量后，需要更新一下

  1. 发现新建项目sync时经常报错，添加仓库源
    配置settings.xml文件中的镜像

  2. 发现还是会有问题，jetty插件会报错。
    据说添加版本号version即可

    清除缓存

    上述方法不行，在plugin中加上version，rebuild一下即可。

  3. 项目结构或设置中JAVA编译器设置错误
    Error:java: error: release version 5 not supported
    解决：
    点击File→ProjectStructure
    确保Project下的SDK版本相同

    确保Modules的Sources中Language level版本与使用的相同

    确保Modules的Dependencies中Module SDKl版本与使用的相同

    打开file->setting，找到java compiler
    确保Target byte code version 与所选版本相同

    rebuild重新执行一下，解决了。
